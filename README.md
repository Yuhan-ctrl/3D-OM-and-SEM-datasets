# 3D-OM-and-SEM-datasets
This repository provides original data and demonstrative correlative analysis for the manuscript entitled 'Serial sectioning SEM for 3D reconstruction and correlative characterisation of graphene nanocomposites' authored by Yuhan Li and Milo Shaffer. Please feel free to use the datasets for modeling, image analysis, or other relevant material studies. 
| Fold names | Description |
| ---------: |-------------|
| OM images sections            | Optical micropic images of nanocomposite sections                                    |
| OM maps                       | Optical concentration maps of nancomposite sections                                  |
| SEM images                    | SEM images of nanocomposite sections at different magnifications                     |
| Correlative analysis example  | Containing OM ans SEM datasets, landmark analysis, and demonstrative images or videos|
